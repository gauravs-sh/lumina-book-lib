# Frontend (React + Vite)

## Overview
React UI for authentication, book/document management, ingestion monitoring, and RAG Q&A.

## Setup
1. Install dependencies:
   - `npm install`
2. Set API base (optional):
   - Create `.env` with `VITE_API_BASE=http://localhost:8000/api/v1`
3. Run dev server:
   - `npm run dev`

## Testing
- `npm test`
