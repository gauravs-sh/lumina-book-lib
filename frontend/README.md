# Frontend (Next.js)

## Overview
SSR-capable UI for authentication, book ingestion, borrowing, reviews, and recommendations.

## Setup
1. Install dependencies:
   - `npm install`
2. Set API base (optional):
   - Create `.env` with `NEXT_PUBLIC_API_BASE=http://localhost:8000/api/v1`
3. Run dev server:
   - `npm run dev`

## Testing
- `npm test`
